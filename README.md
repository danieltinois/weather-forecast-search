# Weather forecast search

## 🚀 Technologies used

- ✅ **React**
- ✅ **Vite**
- ✅ **Axios**
- ✅ **Tailwind CSS**

## 🛠️ How to use

- Enter the name of a city in the search box.
- The weather forecast for the selected city will be displayed, including temperature, weather description, wind chill and humidity.
